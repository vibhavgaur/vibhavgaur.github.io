This website is an experiment that I undertook because I was curious about web development. I wanted to make a website that had the aesthetic of old websites from around the time the internet was born. I liked the minimalistic look of those websites of old, and wanted to create something similar for myself. So, here we are. As such, this website is an active project. Parts of this site will be perpetually “under construction” as I get new ideas about what to do with it. I figured a good purpose for this website would be to host my blog and write-ups about some of the interesting projects that I’ve worked on. Hope you find it interesting!
